## ✨ Features

- ⚡ **Animated Loading Screen** — smooth entry experience
- 🌗 **Dark / Light Theme Toggle** — persisted via localStorage
- 🎨 **Gradient Mesh Background** — dynamic animated background
- 💫 **Particle System** — canvas-based floating particles
- ✍️ **Typing Animation** — cycling role titles
- 🖼️ **Profile Image** — circular with glowing animated border
- 🔢 **Animated Stat Counters** — triggered on scroll
- 📱 **Fully Responsive** — mobile-first design with hamburger menu
- 🃏 **3D Tilt Cards** — interactive hover effects on skill & service cards
- 🖱️ **Mouse Glow Effect** — cursor-following glow
- 📬 **Contact Form** — with validation and feedback messages
- 🔝 **Scroll to Top Button** — appears after scrolling down
- 🎯 **Smooth Scroll Navigation** — active link highlighting

## 🚀 Getting Started

### Prerequisites

No build tools or dependencies required. Just a modern web browser.

### Installation

1. **Clone or download** the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

2. **Navigate** to the project folder:

```bash
cd portfolio
```

3. **Open** `index.html` in your browser:

```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

That's it — no npm install, no build step needed.

## 🎨 Customization

### Update Personal Info

Edit `index.html` to change:
- Your **name**, **bio**, and **description**
- **Contact details** (email, phone, location)
- **Social media links**
- **Stats** (projects, years of experience, clients)

### Update Profile Photo

Replace the image at `images/shreshta img.jpeg` with your own photo, or update the `src` in `index.html`:

```html
<img src="images/your-photo.jpg" alt="Your Name" class="profile-img">
```

### Change Color Theme

Edit the CSS variables in `style.css`:

```css
:root {
    --primary-color: #6366f1;    /* Main purple */
    --secondary-color: #ec4899;  /* Pink accent */
    --accent-color: #3b82f6;     /* Blue accent */
}
```

### Update Typing Roles

Edit the `roles` array in `script.js`:

```js
const roles = [
    'Frontend Developer',
    'Full Stack Developer',
    'UI/UX Designer',
    'React Specialist',
    'Problem Solver'
];
```
